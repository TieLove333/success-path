# Social Media Success Path

A premium Next.js project built with modern design principles, optimized for SEO, and featuring a high-performance architecture.

## 🚀 Features

- **Next.js 15+** with App Router
- **Premium Design System**: Built-in CSS variables for consistent styling
- **Glassmorphism & Gradients**: Modern UI utilities
- **SEO Optimized**: Pre-configured metadata and semantic HTML
- **Responsive Design**: Mobile-first approach
- **Modern Typography**: Outfit and Plus Jakarta Sans fonts

## 🛠️ Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## 📁 Project Structure

- `src/app`: Page components and global styles
- `public`: Static assets
- `src/components`: (Recommended) Reusable UI components
- `src/styles`: (Recommended) Module-specific styles
